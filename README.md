# Zomato Bangalore Restaurant Analysis Dashboard

## Project Overview

This project focuses on analyzing restaurant data from Bangalore using Microsoft Excel. The objective was to identify trends in restaurant distribution, customer ratings, cuisines, pricing categories, and customer engagement. The project demonstrates end-to-end data analysis, including data cleaning, transformation, visualization, and dashboard creation.

## Tools Used

* Microsoft Excel
* Power Query
* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Design

## Dataset Information

The dataset contains information about 6,915 restaurants in Bangalore, including:

* Restaurant Name
* Area
* Rating
* Number of Reviews
* Cuisine Type
* Restaurant Type
* Online Order Availability
* Table Booking Availability
* Average Cost for Two

## Data Cleaning Process

* Removed null and inconsistent values
* Corrected data types
* Cleaned restaurant names
* Created Cost Category groups
* Standardized categorical values
* Prepared data using Power Query

## Analysis Performed

### 1. Top Areas by Number of Restaurants

Identified areas with the highest concentration of restaurants.

### 2. Average Rating by Area

Compared restaurant ratings across different locations.

### 3. Restaurant Type Analysis

Analyzed the distribution of restaurant categories.

### 4. Cuisine Analysis

Identified the most popular cuisines in Bangalore.

### 5. Online Order Analysis

Compared ratings based on online ordering availability.

### 6. Table Booking Analysis

Compared ratings based on table booking availability.

### 7. Cost Category Analysis

Segmented restaurants into different pricing groups.

### 8. Customer Engagement Analysis

Identified restaurants with the highest number of reviews.
